Need to pass API keys to functions?

Functions that call external APIs (OpenAI, Stripe, etc.) can receive secrets via a header. Add X-MCPWorks-Env to your run server config:

"headers": {
  "Authorization": "Bearer YOUR_KEY",
  "X-MCPWorks-Env": "base64:eyJPUEVOQUlfQVBJX0tFWSI6InNrLXh4eCJ9"
}

Encode with: echo -n '{"KEY":"value"}' | base64 — secrets are never stored, only injected at execution time. Full guide →
