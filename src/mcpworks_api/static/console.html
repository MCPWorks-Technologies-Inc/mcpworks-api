Create Agent

An agent is an autonomous entity that can run on a schedule, respond to webhooks, and maintain state.

Destroy Agent

Permanently destroy and all its configuration:
  • All schedules will be removed
  • All webhooks will be removed
  • All stored state will be deleted
  • All channel connections will be removed
This action cannot be undone.
